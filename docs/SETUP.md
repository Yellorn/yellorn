# Yellorn Project Setup Guide

This guide will help you set up the Yellorn project for local development.

---

## Prerequisites
- Python 3.11+
- Node.js 20+
- npm (or yarn/pnpm)
- Git

---

## 1. Clone the Repository
```bash
git clone https://github.com/Yellorn/yellorn.git
cd yellorn
```
